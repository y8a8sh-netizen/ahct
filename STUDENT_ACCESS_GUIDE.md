# 📱 دليل الوصول لصفحة المتدرب

## 🔗 الرابط المباشر

بعد تشغيل المشروع، استخدم الرابط التالي:

```
http://localhost:5173
```

## 📋 خطوات الوصول

### 1️⃣ تشغيل المشروع

```bash
npm install
npm run dev
```

### 2️⃣ الدخول إلى النظام

1. افتح المتصفح
2. اذهب إلى: `http://localhost:5173`
3. سيظهر لك صفحة تسجيل الدخول
4. اختر **"متدرب"** من القائمة
5. أدخل الرقم التدريبي

### 3️⃣ البحث عن الجدول

بعد تسجيل الدخول:
- انقر على **"بوابة المتدربين"**
- أدخل الرقم التدريبي في مربع البحث
- اضغط **"بحث"**
- سيظهر جدول الاختبارات الخاص بك

## 📱 استخدام الجوال

النظام الآن **متجاوب بالكامل** ويعمل على:
- ✅ الجوال (Mobile)
- ✅ التابلت (Tablet)
- ✅ الكمبيوتر (Desktop)

### مميزات النسخة المحمولة:
- قائمة منسدلة في الجوال
- جداول قابلة للتمرير أفقياً
- نصوص وأزرار محسّنة للشاشات الصغيرة
- تجربة مستخدم سلسة

## 🔧 استكشاف الأخطاء

### المشروع لا يعمل بعد النقل

1. **تثبيت التبعيات:**
```bash
npm install
```

2. **تشغيل المشروع:**
```bash
npm run dev
```

3. **التحقق من المنفذ:**
- إذا كان المنفذ 5173 مشغولاً
- سيظهر لك رقم منفذ آخر
- استخدم الرابط الذي يظهر في Terminal

### الجوال لا يظهر التصميم الصحيح

1. **امسح الكاش:**
   - Chrome: Settings → Privacy → Clear browsing data
   - Safari: Settings → Safari → Clear History

2. **تحديث الصفحة:**
   - اضغط `Ctrl + F5` (Windows)
   - أو `Cmd + Shift + R` (Mac)

## 📊 روابط أخرى

- **لوحة تحكم المدير**: http://localhost:5173 (دخول كمدير)
- **بوابة المراقبين**: http://localhost:5173 (دخول كمراقب)
- **رئيس القسم**: http://localhost:5173 (دخول كرئيس قسم)
